# Pokémon or Medicine?

A browser-based guessing game built with HTML, CSS, and vanilla JavaScript.

## How to Run

1. Open `index.html` in a web browser.
2. Read the displayed name.
3. Click `Pokémon` or `Medicine`.
4. Review the feedback, then click `Next Round`.
5. Click `Restart Game` to reset score and rounds played.

## Files

- `index.html` contains the game layout.
- `styles.css` contains the visual styling.
- `script.js` contains the game data and logic.
- `README.md` contains usage instructions.
